This project, titled Log Correlation & Real-Time Attack Detection System, is designed to build a centralized platform for analyzing large volumes of log data and detecting security threats in real time. In modern systems, logs contain valuable information related to system activities and potential attacks, but manually monitoring these logs is inefficient and time-consuming. To address this problem, this project automates the process of log collection, parsing, analysis, and visualization, enabling faster and more effective security monitoring.

The system works by allowing users to upload log files, which are then parsed into a structured format for better analysis. These processed logs are stored in a MySQL database, where they are further examined by a rule-based detection engine. This engine identifies suspicious patterns and detects various types of attacks such as brute force attacks (repeated failed login attempts), SQL injection (malicious database queries), cross-site scripting (XSS) attacks (injection of harmful scripts), and directory traversal attacks (unauthorized access to restricted directories). Once an attack is detected, the system generates alerts based on severity levels, helping users quickly understand the level of threat.

To make the analysis more intuitive, the project includes a real-time dashboard built using HTML, CSS, and Chart.js, which visually represents detected attacks and log data. The backend is developed using Python and Django, ensuring efficient processing and scalability. Overall, this system simplifies security analysis by automating log monitoring, improving threat detection speed, and providing clear visual insights, making it a useful solution for enhancing cybersecurity in modern applications.
